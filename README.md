# DashBoard

## Installation

Clone project folder 

```bash
git clone https://github.com/narasimhan26/Dashboard.git
```

Install NPM packages

```bash
npm install
```
Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
